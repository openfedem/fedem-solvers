enums module

This module contains some enum definitions for the fedempy package. They are python equivalents to corresponding enum definitions in the C++ code of the FedemDB shared object library (see the fedem_mdb repository). The numerical value of each enum value in this module should therefore not be changed without a similar change in the C++ equivalent, to preserve consistency.
